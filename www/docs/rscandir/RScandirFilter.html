Class: RScandirFilter

RScandirFilter

Filter object allows wildcard, RegEx, minimatch pattern matching.

new RScandirFilter(filelist, filter)

Parameters:
Name Type Description
filelist Array The list of files to filter.
filter String | RegExp | Object If is a string, is a filter for wildstring/minimatch. If is a RegExp, performs a match. Exclude option may be set as an object.
