Class: FileHelpers

FileHelpers

Helpers container class for Node.js filesystem. Includes finding first non-node_modules folder, the original executed file, the current functions parent file, all packages in a directory, resolving packageName/file paths.

<static> getMainDirectory()

Heuristic for guessing main directory. This includes resolving the following use cases: 1) ./node_modules/.bin scripts 2) ./node_modules/PACKAGE_NAME/file scripts 3) ./SUBDIR/file calls The first non-node_modules folder with a package.json in it is used.
