Class: Override

Override

Override allows functions to be overridden. If need be, overridden functions can be, "released" and restored to their original state as well.

new Override(originalFn, newFn, context)

Parameters:
Name Type Default Description
originalFn String The string name of the function. This name gets eval'd to acquire the reference.
newFn function null The function reference to overwrite the original with.
context Object null (Optional) The context. If omitted the context is grabbed from the original function. If no context can be found, a null context is used.

Methods

<static> releaseAll()

Releases all functions from their overrides.

<static> resetAll()

Resets all overrides.

release()

Releases the override, destroying references to the object and preventing all further overrides from occurring.

reset()

Resets the wrapper.

set()

Sets the new override function.

unwrap()

Unwraps a layer off the chain and then sets the function to the next chain item.

wrap(newFn)

Wraps the original function with the newFn. If originalFn has already been wrapped, then wraps the most recent wrapper function.
Parameters:
Name Type Description
newFn function The new function that will always be called with the chain function as the first parameter.
