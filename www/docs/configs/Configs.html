Class: Configs

Configs

Wraps the Files object to provide more articulated control over JSON-based configuration files. Allows defaults and overrides to be used.

readCLI()

Reads command-line for inputs. If an argument maps to an existing config value, the config value is overwritten. Must specify command-line argument to address PACkAGE_NAME['option']=VALUE. By default, if no subscript exists, then the package name of the executing package is used unless a config key is specified in the executing package's package.json file.
