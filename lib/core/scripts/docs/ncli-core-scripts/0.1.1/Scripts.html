Class: Scripts

Scripts

Class transposes files from scripts/* path and adds them to active package.json on installation. Scripts may have the same name. Lifecycle scripts are run through this package and then initiated on other packages. As a result, lifecycle script execution order is maintained. If you need to maintain dependency chain order for lifecycle scripts, it's recommended to utilize NPM hooks directly. Set the `ignoreLifecycleScripts` to include an entry either with the packageName or with the package/script format.
